## 📋 404 页面设计特点分析

### 视觉元素
1. **大型动画数字/图标** - 120px 超大渐变文字
2. **浮动动画** - 轻柔的上下浮动 + 旋转
3. **波浪装饰** - 三层流动波浪效果
4. **渐变色彩** - accent → accent-2 渐变
5. **发光效果** - 文字/元素阴影
6. **分层动画** - 错开的淡入效果
7. **卡片式设计** - 圆角卡片 + 内阴影
8. **大按钮** - 44px 高度，圆角，图标+文字
9. **友好文案** - 轻松、亲切的语气

### 交互特点
1. **Hover 上浮** - translateY(-2px)
2. **增强阴影** - 多层阴影叠加
3. **平滑过渡** - 200ms cubic-bezier
4. **视觉反馈** - 清晰的状态变化

## 🎨 设计系统

### 字体大小
- **超大标题**: 64px（页面主标题）
- **大标题**: 48px（区块标题）
- **指标数字**: 72px（统计数据）
- **卡片标题**: 24px
- **正文**: 16px
- **辅助文字**: 14px

### 间距系统
- **超大间距**: 48px
- **大间距**: 32px
- **中等间距**: 24px
- **标准间距**: 16px
- **小间距**: 12px
- **微小间距**: 8px

### 圆角
- **超大圆角**: 24px（卡片）
- **大圆角**: 16px（弹窗）
- **标准圆角**: 12px（按钮）
- **小圆角**: 8px（输入框）
- **圆形**: 999px（标签）

### 阴影层次
- **浮动**: 0 8px 32px rgba(0,0,0,0.12)
- **悬浮**: 0 12px 48px rgba(0,0,0,0.16)
- **发光**: 0 0 24px rgba(31,201,160,0.3)

## ⚠️ 注意事项

### 保持一致性
- 所有大按钮统一 44px 高度
- 所有卡片统一圆角 24px
- 所有动画统一 200-300ms 时长

### 性能考虑
- 动画使用 transform（GPU 加速）
- 避免过多同时动画
- 尊重 prefers-reduced-motion

### 无障碍
- 保持足够对比度
- 保留键盘导航
- 保留屏幕阅读器支持
