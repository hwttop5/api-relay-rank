# 🌙 暗色主题优化分析

## 📋 当前问题分析

### 1. 对比度问题
**问题**: 一些文本对比度不够
- `--muted: #a5b3ad` 在深色背景上可能不够清晰
- `--line: #303940` 分隔线太暗

**建议**:
- 提升 muted 文字亮度
- 增强分隔线对比度

### 2. 卡片/面板层次感不足
**问题**: 背景色过于接近
- `--bg: #0d0f12`
- `--panel: #171b20`
- `--panel-soft: #1d2329`

**建议**:
- 增加面板与背景的对比度
- 添加更明显的内发光效果

### 3. 阴影效果不明显
**问题**: 深色主题下阴影难以察觉
- `--shadow: 0 24px 70px rgba(0, 0, 0, 0.36)`
- 黑色阴影在深色背景上不明显

**建议**:
- 使用带颜色的阴影（青绿色）
- 增加发光效果替代传统阴影

### 4. 按钮状态不够突出
**问题**: hover 状态变化不明显
- 需要更强的视觉反馈

**建议**:
- 增强 hover 时的发光效果
- 添加更明显的颜色变化

### 5. 表格可读性
**问题**: 表格行背景色对比度低
- `--row-bg: rgba(18, 22, 26, 0.62)`
- `--row-bg-alt: rgba(24, 29, 34, 0.62)`

**建议**:
- 增加行背景透明度差异
- 增强 hover 状态

### 6. 输入框可见性
**问题**: 输入框边框不够明显
- `--line: #303940` 太暗

**建议**:
- 增加输入框边框亮度
- 添加微妙的发光效果

### 7. 顶部导航栏
**问题**: 与背景融合度过高
- `--topbar-bg: rgba(15, 18, 22, 0.86)`

**建议**:
- 增加边框发光
- 提升背景亮度

### 8. 渐变色不够明显
**问题**: 渐变标题在深色背景上不够突出

**建议**:
- 增加渐变对比度
- 添加文字发光效果

---

## 🎨 优化方案

### 方案 1: 提升对比度
- 提亮文字颜色
- 增强分隔线
- 明确面板层次

### 方案 2: 增强发光效果
- 为重要元素添加发光
- 使用青绿色发光替代黑色阴影
- 增强 hover 状态发光

### 方案 3: 优化层次感
- 拉大背景色差异
- 增强内发光效果
- 添加微妙的边框发光

### 方案 4: 改善表格
- 增加行背景对比度
- 增强 hover 状态
- 添加边框高亮

### 方案 5: 优化输入框
- 提升边框亮度
- 添加聚焦发光
- 增强占位符可见性
